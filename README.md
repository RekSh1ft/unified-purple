# Unified Purple Theme 

I made this theme because there was a theme called *Amethyst* hasn't been updates in 4 years (2022). I took inspiration from this theme and another theme called *Midnight*

---

## Features

* Dark purple Amethyst UI
* Custom accent colors
* Lightweight CSS overrides

---

## Customization

You can edit the following in `midnight.css`:

### Background colors

```css
--bg-4: #1e1628; /* darkest background */
--bg-3: #241a31;
--bg-2: #2b1f3a;
--bg-1: #322443; /* lighter panels */
```

### Accent colors

```css
--accent-1: #c9a3ff;
--accent-2: #b07cff;
--accent-3: #9a5cff;
--accent-4: #7f3cff;
--accent-5: #6626ff;
```
